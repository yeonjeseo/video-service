# 비디오 분할 저장 / 분할 영상 제공 API

## 기술스택

- Node v20.10 LTS
- express v4.18.2
- MariaDB v11
- fluent-ffmpeg v2.1.2

## 사전요구사항 

- Docker version 24.0.6
  - 이하 버전에서는 확인하지 않았음.

## 실행 방법

- `git clone https://github.com/yeonjeseo/video-service.git`
- 
- `docker-compose up`

## API 명세

- `localhost:3000/docs`
